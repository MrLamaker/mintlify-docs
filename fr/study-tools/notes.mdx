Les notes dans Clamly vous offrent un éditeur Markdown plein écran sans barres latérales ni barres d'outils. Rédigez des résumés de cours, des plans de dissertation ou des guides d'étude, puis organisez le tout avec des épinglages et des étiquettes de couleur.

## Créer une note

<Steps>
  <Step title="Ouvrez la section Notes">
    Cliquez sur **Notes** dans la barre latérale puis sur **Nouvelle note**. Un éditeur vierge s'ouvre immédiatement.
  </Step>
  <Step title="Ajoutez un titre">
    Tapez un titre en haut de l'éditeur. Ce sera le nom de la note dans votre bibliothèque, alors rendez-le suffisamment descriptif pour la retrouver plus tard.
  </Step>
  <Step title="Rédigez votre contenu">
    Commencez à écrire dans la zone d'éditeur sous le titre. L'éditeur prend en charge le Markdown complet. Voir les [Astuces de mise en forme](#formatting-tips) ci-dessous.
  </Step>
  <Step title="Enregistrez">
    Les notes s'enregistrent automatiquement à mesure que vous tapez. Vous pouvez aussi utiliser le raccourci d'enregistrement ou cliquer sur **Enregistrer** dans la barre d'outils à tout moment.
  </Step>
</Steps>

## L'éditeur plein écran

Cliquez sur l'icône **plein écran** dans le coin supérieur droit d'une note pour entrer en mode écriture sans distraction. En plein écran, la barre latérale et la navigation disparaissent, ne laissant que votre texte. Appuyez sur **Échap** ou cliquez sur l'icône de sortie pour revenir à la vue normale.

<Tip>
  Le mode plein écran se marie bien avec le minuteur de concentration. Démarrez une session Pomodoro, passez en plein écran sur vos notes et travaillez sans distractions jusqu'à la fin du minuteur.
</Tip>

## Astuces de mise en forme

<CardGroup cols={2}>
  <Card title="En-têtes" icon="heading">
    Utilisez `#` pour un titre de premier niveau, `##` pour un sous-titre et `###` pour une section plus petite.
  </Card>
  <Card title="Gras et italique" icon="bold">
    Encadrez le texte de `**double astérisques**` pour le **gras** et de `*simples astérisques*` pour l'*italique*.
  </Card>
  <Card title="Listes" icon="list">
    Commencez une ligne par `-` ou `*` pour une liste à puces. Utilisez `1.` pour une liste numérotée. Indentez de deux espaces pour créer des sous-listes.
  </Card>
  <Card title="Blocs de code" icon="code">
    Encadrez le code en ligne d'apostrophes inverses : `` `nomDeVariable` ``. Pour du code multi-lignes, utilisez trois apostrophes inverses (` ``` `) sur des lignes séparées au-dessus et en dessous du bloc.
  </Card>
</CardGroup>

## Épingler une note

Épinglez les notes importantes pour qu'elles apparaissent toujours en haut de votre bibliothèque, peu importe le nombre de nouvelles notes que vous ajoutez.

<Steps>
  <Step title="Ouvrez le menu de la note">
    Survolez la note dans votre bibliothèque et cliquez sur l'icône **⋯** (plus d'options), ou ouvrez la note et cherchez l'icône d'épingle dans la barre d'outils.
  </Step>
  <Step title="Épinglez la note">
    Cliquez sur **Épingler en haut**. La note apparaîtra désormais au-dessus de toutes les notes non épinglées avec un indicateur d'épingle pour la repérer facilement.
  </Step>
</Steps>

## Coder vos notes par couleur

Attribuez une étiquette de couleur à n'importe quelle note pour bâtir un système d'organisation visuel dans votre bibliothèque.

<Steps>
  <Step title="Ouvrez le menu de la note">
    Survolez la note et cliquez sur l'icône **⋯**, ou accédez au sélecteur de couleur depuis l'intérieur de la note ouverte.
  </Step>
  <Step title="Choisissez une couleur">
    Sélectionnez une couleur dans la palette. La carte de la note dans votre bibliothèque reflétera la couleur immédiatement.
  </Step>
</Steps>

Une approche courante : une couleur par matière. Bleu pour la chimie, vert pour l'histoire, jaune pour tout ce qui nécessite un suivi.

## Exporter une note

<Steps>
  <Step title="Ouvrez la note">
    Naviguez vers la note que vous souhaitez exporter.
  </Step>
  <Step title="Ouvrez les options d'export">
    Cliquez sur le menu **⋯** ou l'icône d'export dans la barre d'outils et sélectionnez **Exporter**.
  </Step>
  <Step title="Choisissez un format">
    Sélectionnez le format d'export préféré et confirmez. Le fichier sera téléchargé sur votre appareil immédiatement.
  </Step>
</Steps>

<Note>
  Les notes exportées conservent toute la mise en forme Markdown, vous pouvez donc les ouvrir dans n'importe quel éditeur compatible ou les importer dans un autre outil d'écriture sans perdre votre structure.
</Note>
