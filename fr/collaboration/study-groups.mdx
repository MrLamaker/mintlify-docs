Les groupes d'étude vous offrent un espace partagé pour distribuer des quiz et vous coordonner avec vos camarades. Créez un groupe en quelques secondes, partagez votre code d'invitation et commencez à partager du matériel d'étude.

## Créer un groupe d'étude

<Steps>
  <Step title="Ouvrez l'onglet Groupes">
    Naviguez vers la section **Collaboration** dans la barre latérale gauche et sélectionnez **Groupes d'étude**.
  </Step>
  <Step title="Créez un nouveau groupe">
    Cliquez sur **Créer un groupe**, entrez un nom de groupe et ajoutez éventuellement une courte description.
  </Step>
  <Step title="Partagez votre code d'invitation">
    Une fois le groupe créé, Clamly génère un **code d'invitation** unique. Partagez ce code avec vos camarades. Ils l'entreront pour rejoindre.
  </Step>
</Steps>

## Rejoindre un groupe

<Steps>
  <Step title="Allez sur Groupes d'étude">
    Ouvrez la page **Groupes d'étude** depuis la section Collaboration.
  </Step>
  <Step title="Entrez le code d'invitation">
    Cliquez sur **Rejoindre un groupe** et saisissez le code d'invitation que vous avez reçu. Clamly vous ajoutera immédiatement au groupe en tant que **Membre**.
  </Step>
</Steps>

## Rôles et permissions

Chaque groupe d'étude comporte trois rôles. Chacun confère un niveau d'accès différent.

<CardGroup cols={3}>
  <Card title="Propriétaire" icon="crown">
    La personne qui a créé le groupe. Contrôle complet : gestion des membres, attribution ou retrait du rôle Admin, partage de quiz et suppression du groupe.
  </Card>
  <Card title="Admin" icon="shield">
    Peut inviter de nouveaux membres, gérer les membres existants et partager des quiz dans le groupe.
  </Card>
  <Card title="Membre" icon="user">
    Peut consulter et interagir avec les quiz partagés et participer à l'activité du groupe. Ne peut pas gérer les membres ni modifier les paramètres du groupe.
  </Card>
</CardGroup>

## Gérer les membres

<Accordion title="Promouvoir un membre en Admin">
  Ouvrez la liste **Membres** du groupe, trouvez le membre à promouvoir et sélectionnez **Nommer Admin**. Il obtiendra immédiatement les permissions de niveau admin.
</Accordion>

<Accordion title="Retirer un membre du groupe">
  Dans la liste **Membres**, sélectionnez le membre puis **Retirer du groupe**. Il perdra immédiatement l'accès à tout le contenu du groupe et aura besoin d'un nouveau code d'invitation pour revenir.
</Accordion>

<Accordion title="Transférer la propriété">
  Si vous souhaitez céder le rôle de Propriétaire, sélectionnez un Admin actuel puis **Transférer la propriété**. Votre rôle deviendra Admin après le transfert.
</Accordion>

## Partager des quiz dans le groupe

<Steps>
  <Step title="Ouvrez un quiz">
    Allez dans votre bibliothèque **Quiz** et ouvrez le quiz que vous souhaitez partager.
  </Step>
  <Step title="Partagez avec le groupe">
    Cliquez sur le bouton **Partager** et sélectionnez **Partager avec un groupe**. Choisissez le groupe dans votre liste et confirmez.
  </Step>
  <Step title="Les membres obtiennent l'accès">
    Tous les membres actuels du groupe peuvent désormais consulter et utiliser le quiz partagé. Tout membre qui rejoint plus tard le verra également dans le contenu partagé du groupe.
  </Step>
</Steps>

## Suivre l'activité du groupe

Depuis la page principale du groupe, vous pouvez voir les quiz récemment partagés, les nouveaux membres qui ont rejoint, et l'activité générale dans le fil du groupe.

## Utilisation acceptable

<Note>
  Vous êtes responsable du contenu que vous partagez dans un groupe d'étude. Assurez-vous que tout ce que vous publiez respecte les [Conditions d'utilisation de Clamly](https://clamly.app/terms). Les groupes ou contenus qui enfreignent les conditions peuvent être retirés par Clamly à tout moment.
</Note>

<Tip>
  Gardez votre code d'invitation privé et ne le partagez qu'avec des personnes de confiance. Si vous pensez que votre code a été partagé trop largement, les Propriétaires et Admins peuvent retirer tout membre qui ne devrait pas avoir accès.
</Tip>
