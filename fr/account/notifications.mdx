Clamly utilise des notifications push pour garder vos sessions d'étude sur la bonne voie. Activez celles que vous souhaitez. Toutes les notifications sont sur consentement.

## Types de notifications

<CardGroup cols={3}>
  <Card title="Rappels d'étude" icon="clock">
    Recevez un rappel lorsqu'il est temps de commencer une session d'étude planifiée, pour ne jamais laisser passer une session.
  </Card>
  <Card title="Alertes d'examen" icon="bell-ringing">
    Recevez des alertes à l'approche des examens, ce qui vous laisse le temps de réviser et de vous préparer.
  </Card>
  <Card title="Mises à jour de réalisations" icon="trophy">
    Découvrez à l'instant où vous débloquez une nouvelle réalisation ou atteignez un jalon d'étude.
  </Card>
</CardGroup>

## Activer les notifications push

Les notifications push nécessitent une autorisation unique dans votre navigateur.

<Steps>
  <Step title="Ouvrez votre page de profil">
    Cliquez sur votre avatar ou vos initiales dans le coin supérieur droit, puis sélectionnez **Profil** dans le menu déroulant.
  </Step>
  <Step title="Allez dans les paramètres de notification">
    Faites défiler jusqu'à la section **Notifications** sur votre page de profil.
  </Step>
  <Step title="Activez les notifications">
    Activez les types de notification que vous souhaitez recevoir. Si votre navigateur n'a pas encore accordé la permission, une invite apparaîtra vous demandant d'**Autoriser** les notifications de Clamly.
  </Step>
  <Step title="Cliquez sur Autoriser dans votre navigateur">
    Dans l'invite du navigateur, cliquez sur **Autoriser**. Clamly confirmera que les notifications sont désormais actives.
  </Step>
</Steps>

<Note>
  Les notifications push sont diffusées via votre navigateur. Elles fonctionnent lorsque Clamly est ouvert dans un onglet, et dans la plupart des navigateurs modernes, elles peuvent également apparaître lorsque Clamly s'exécute en arrière-plan.
</Note>

## Gérer les notifications depuis votre profil

<Steps>
  <Step title="Allez dans Profil puis Notifications">
    Ouvrez la section **Notifications** sur votre page de profil.
  </Step>
  <Step title="Activez ou désactivez chaque type">
    Utilisez les bascules individuelles pour activer ou désactiver indépendamment les **Rappels d'étude**, les **Alertes d'examen** et les **Mises à jour de réalisations**.
  </Step>
  <Step title="Enregistrement">
    Les changements sont enregistrés automatiquement à mesure que vous activez chaque paramètre.
  </Step>
</Steps>

<Tip>
  Si vous voulez uniquement les alertes d'examen pendant la période d'examens, désactivez simplement les rappels d'étude et les mises à jour de réalisations.
</Tip>

## Désactiver les notifications via les paramètres du navigateur

Si vous souhaitez arrêter toutes les notifications Clamly au niveau du navigateur, vous pouvez révoquer la permission directement dans les paramètres de votre navigateur.

<Accordion title="Google Chrome">
  Allez dans **Paramètres → Confidentialité et sécurité → Paramètres des sites → Notifications**. Trouvez `clamly.app` dans la liste et définissez-le sur **Bloquer**.
</Accordion>

<Accordion title="Mozilla Firefox">
  Allez dans **Paramètres → Vie privée et sécurité → Permissions → Notifications → Paramètres**. Trouvez `clamly.app` et changez le statut en **Bloquer**.
</Accordion>

<Accordion title="Safari (macOS)">
  Allez dans **Safari → Réglages → Sites web → Notifications**. Trouvez `clamly.app` et définissez-le sur **Refuser**.
</Accordion>

<Accordion title="Microsoft Edge">
  Allez dans **Paramètres → Cookies et autorisations de site → Notifications**. Trouvez `clamly.app` dans la liste Autoriser et supprimez-le, ou ajoutez-le à la liste Bloquer.
</Accordion>

<Warning>
  Révoquer la permission au niveau du navigateur désactive toutes les notifications push de Clamly, quels que soient vos réglages dans l'application. Pour les réactiver plus tard, vous devrez redonner la permission depuis votre page de profil.
</Warning>
